# Naruto Shippuden Calculator

A web-based calculator built using Java Servlets, JSP, Maven, and Apache Tomcat with a Naruto Shippuden themed interface.

## Features
- Expression-based calculations
- BODMAS support
- Division by zero handling
- Naruto-themed UI
- Character-themed calculator buttons
- Responsive design

## Technologies
- Java 21
- JSP
- Servlets
- Maven
- HTML
- CSS
- JavaScript
- Apache Tomcat 10

## How to Run
1. Clone the repository.
2. Import as a Maven Project.
3. Configure Apache Tomcat.
4. Run the project.
